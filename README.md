# 🏙️ mOveOn

## 👨‍💻 프로젝트 소개
**무브온(MoveOn)** 은  
1인가구, 신혼부부, 직장발령자 등 **이사 후 지역 정착을 위한 공공데이터 기반 생활 정보 통합 플랫폼**입니다.

---

## 🧩 프로젝트 기획
> “1인가구 증가 + 정보격차 + 안전불안”은 새로운 사회문제로 부상하고 있습니다.  
> 신뢰성 있는 **공공데이터를 활용한 지역 정착 정보 통합 플랫폼**을 통해  
> 정보탐색의 피로감을 줄이고, 기존 **1주일 이상 걸리던 탐색 과정을 2시간~1일 이내로 단축**하여  
> 지역 정착 속도를 높이는 **생활정보 통합 서비스 ‘무브온(MoveOn)’**의 필요성이 제기되었습니다.

---

## 🎯 프로젝트 기대 효과
1. 💡 **공과금 정산부터 안전·교통·생활정보까지** 앱 하나로 해결  
2. 🤖 **공공데이터 + AI + GPS** 결합을 통한 맞춤형 생활 서비스 및 데이터 기반 비즈니스 모델 창출  
3. 🚶‍♀️ **교통 약자 지원**, **안전한 주거 환경**, **스마트한 쓰레기배출 알림 서비스**  
4. 📍 사용자 **위치/주거 정보 기반의 최적화된 맞춤 정보 제공**

## ⚙️ 주요 기능

### 🗓 생활 정보
- 위치기반 **날씨 정보 (공공데이터 API 연동)**  
- **쓰레기 배출정보**  
- **의류수거함 위치정보**  
- **전동휠체어 충전소 지도**  
- **관공서 위치정보**  
- **심야약국 / 병원 위치정보**

### 🛡 안전 정보
- **성범죄자 위치정보**  
- **대피소 위치정보**  
- **공중화장실 위치정보**

### 🚇 교통 정보
- **지하철 승강기/엘리베이터 상태 조회**  
- **지하철 배차 시간 확인**

### 💬 커뮤니티
- **소분 모임 (Group Chat 기능)**  
- **지역 기반 채팅방**
