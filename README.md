# 👨‍💻 프로젝트 소개 
1인가구, 신혼부부,직장발령자등의 이사후 지역 정착을 위한 공공데이터기반 생활 정보 통합 플랫폼

###  프로젝트 기획 
"1인가구증가+정보격차+안전불안"은 새로운 사회문제로 부상하고 있으며, 빠르고 스트레스없는 지역정착라는 사회적 필요를 해결하기위해 신뢰성있는 공공데이터를 활용한 하나의 플랫폼에서 지역정착 정보를 통합 제공함으로써, 정보탐색의 피로감을 줄이고 기존 1주일 이상 걸리던 탐색 과정을 2시간~1일 이내로 단축하여 지역정착속도를 높일수 있는 생활 정보 통합 플랫폼 ‘무브온(Move온)’의 필요성이 제기되었다.

### 프로젝트 기대 효과
1. 공과금 정산부터 안전·교통·생활 정보까지 앱 하나로 해결
2. 공공데이터, AI, GPS를 결합해 맞춤형 생활 서비스 와 데이터 기반 비즈니스 모델 창출
3. 교통 약자 지원, 안전한 주거 환경, 스마트한 쓰레기배출정보 알림
4. 사용자정보(위치,집)기반의 최적정보로 기반으로 새로운 정보창출
